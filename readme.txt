
UI 프로그램 변경 사항
날짜 : 2015.3.12
작성자 : 김현재

1. Video 화면 수정
=============================================================================
: 상태 확인 창 열릴 때에는 비디오 요쳥 안함, 상태 확인 값이 Normal 상태가 될
경우에 Video 요청하도록 변경
=============================================================================

2. 팝업 창 제목 네비게이션 수정
=============================================================================
: Log On Password, Validate Configuration Password, Program Password 에 대한
팝업 창 제목 네비게이션 수정
=============================================================================

3. Mode Name Read
=============================================================================
: UI 프로그램 초기 구동 시, Avr에서 Mode Read 후 UI Mode Name 초기화, 이후에는
UI Server Buffer에 저장
: Mode Name Data Write 는 UI Server, Avr 두 곳 모두에 저장하도록 작성
=============================================================================

4. HELP 창 - 무선 IP Address Read
=============================================================================
: 무선 IP Device를 wlan0 ~ wlan9 까지 모두 읽고 표시하도록 변경 및 확인
: HELP 창에 진입할 시에 유, 무선 IP Address에 대한 정보를 표시함
=============================================================================
